# Learning Spark
## Implementations:
- ### Spark Streaming
    - ### Count Words
        - ### Run ```nc -lk 5001``` and type the words separeted by " "
        - ### Run ```python3 count_words.py```
    - ### Read Json Stream
        - ### Run ```nc -lk 5001``` and type json format ```{"Symbol": "XXXXXX", "Timestamp": 1290460517, "Price": 2223.05000000}```
        - ### Run ```python3 read_json_stream.py```
